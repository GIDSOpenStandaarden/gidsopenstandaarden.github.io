Afsprakenstelsel : SamenBeter versie 0.2 SNS - Launch

Created by Remy Lamers, last modified on jul 04, 2019

Beste lezer,

Voor je ligt de beta versie (0.2) van SamenBeter Sociale Netwerk Standaard (SNS). Als onderdeel van deze beta versie van de SNS levert SamenBeter een protocol en een set van afspraken op.

Launch
Dit eerste protocol, de Launch, zorgt ervoor dat portaal applicaties externe applicaties, zoals behandelingen (zelfzorg-modules), games en tools naadloos kunnen integreren. Dit protocol maakt gebruik van JSON Webtokens (JWT).
Dit protocol is als eerste ontwikkeld omdat de vraag vanuit de wijken het meest concreet was om drempelloos zelf-zorg modules te kunnen benaderen vanuit een wijkplatform. 

Dit protocol biedt enkele voordelen voor zowel wijkbewoners als aanbieders van zorgmodules:

  • Een naadloze integratie houdt voor wijkbewoners in dat ze moeiteloos en veilig met hun profiel vanuit het voor hen vertrouwde wijkplatform, een zelfzorg-module van een zorgaanbieder kunnen volgen.
    Ze hoeven hiervoor geen fysieke account aan te maken op een zorgplatform.
  • Een naadloze integratie houdt voor jou als aanbieder een lichte implementatie in, die binnen een dag te realiseren is. Daarnaast bereik je zo met jouw aanbod van zelfzorg modules een groter publiek, terwijl wel herleidbaar blijft waar deze gebruikers vandaan komen.
